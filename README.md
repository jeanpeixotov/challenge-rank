Api Rank
=========

This API gets the top five apps from each google play category.


## Technologies

* Gulp for compile task
* Docker
* Knex (Database Code First)

### Back-End
* Node
* Typescript
* Postgres (Knex + Objection.js) 

## For Dev

```bash
# install depedencies
yarn

#run
docker-compose up
```
